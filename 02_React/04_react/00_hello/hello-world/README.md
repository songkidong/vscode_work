# (최초1번) npm install -g create-react-app

# react 프로젝트 만들기 : 
npx create-react-app 프로젝트명

# react 프로젝트 실행하기 :
cd 프로젝트명 (프로젝트명 폴더로 들어가기) cd:(change directory)
npm start (react 서버 가동 명령어)

# react 프로젝트 중단하기 :
vscode 터미널창에서 : ctrl + c

# react 화면 보기 (웹 브라우저 주소창에 아래 입력)
http://localhost:3000
